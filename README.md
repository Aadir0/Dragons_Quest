# Dragons_Quest
## My First 2-D Platformer game
![Banner](Assets/Sprites/gemini-3-pro-image-preview-2k_(nano-banana-pro)_a_Remove_this_New_Game.png)

### This is a game where you have to push yourself and run as far as you can go by clearing traps and killing some wandering enemys.
## Key Binds:
- W/Left Arrow - Move Left
- D/Right Arrow - Move Right
- Space - Jump
- Left Mouse Click - Fire

## Hope You will enjoy this game!!
